# Todo app

A multi user todo app powered by modern technologies.

### Run the app

 #### `docker-compose -p todo-app -f docker/docker-compose.yml --env-file .env.example up`

## Technologies

 ### [Frontend](./frontend/)

  - `Svelte`
  - `Bulma`

 ### [Backend](./api/)

  - `Postgres`
  - `Fastify`
  - `JWT`

 ### [Putting it all together](./docker/)

  - `Docker`
  - `docker-compose`
